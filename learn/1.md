## Day 2：Node.js 调用 CEX 行情接口

初始化项目

```bash
mkdir cex-node && cd cex-node
npm init -y
npm i axios ccxt dotenv
```


⸻

使用 ccxt（强烈推荐）

ccxt 是 事实标准

示例：获取 BTC 行情

```js
import ccxt from 'ccxt'

const exchange = new ccxt.binance()

const ticker = await exchange.fetchTicker('BTC/USDT')
console.log(ticker.last)
```

📌 输出：
	•	最新价格
	•	成交量
	•	24h 高低


⸻

## Day 3：账户接口（只用测试环境）

⚠️ 一定用测试网 / 模拟账户

配置环境变量

```js
API_KEY=xxx
API_SECRET=yyy

const exchange = new ccxt.binance({
  apiKey: process.env.API_KEY,
  secret: process.env.API_SECRET,
})

```

查询账户余额

```js
const balance = await exchange.fetchBalance()
console.log(balance.total)
```

📌 理解重点：
	•	CEX 余额 ≠ 链上余额
	•	是数据库余额

## 🗓️ Day 6：充提 & CEX vs DEX 本质区别

提币流程（核心）

数据库扣余额 → 生成交易 → 上链 → 等确认

| 维度 | CEX | DEX |
| --- | --- | --- |
| 资产 | 平台托管 | 自托管 |
| 撮合 | 服务器 | 智能合约 |
| 速度 | 快 | 慢 |
| 风险 | 平台风险 | 合约风险 |
